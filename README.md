# 🚀 AutoML ML Engine

An end-to-end Machine Learning pipeline built from scratch using Python.
This project automates data ingestion, preprocessing, model training, and evaluation.

---

## 📌 Features

* 📥 Data Ingestion (CSV / URL support)
* 🧹 Data Preprocessing Pipeline
* 🤖 Multi-model Training (Linear, Ridge, Lasso, XGBoost)
* 📊 Model Evaluation & Comparison
* ⚡ Scalable & Modular Architecture

---

## ⚙️ Installation

```bash

cd automl-ml-engine

python -m venv venv
venv\Scripts\activate   # Windows

```
